# Quiz

Full Stack Quiz Application built on HonoJS, ReactJS, Bun, Typescript.

![Quiz](./public/Quiz-Home.png?raw=true "quiz-home")

## Pre-requisites
* Bun runtime

## To build locally
```bash
git clone https://github.com/Nishaaaanth/Quiz
```

```bash
bun i
```

## To run locally
```bash
bun run dev
```
